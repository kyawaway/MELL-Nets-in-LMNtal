# MELL-Nets-in-LMNtal

This is the part of the presentation at [APLAS2023](https://conf.researchr.org/details/aplas-2023/src-and-posters/4/-Non-SRC-Encoding-MELL-Cut-Elimination-into-a-Hierarchical-Graph-Rewriting-Language).
